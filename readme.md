# `vmini-build`
> 配合 `@vue-mini/wechat` 使用的 `vue sfc` 编译器 + 脚手架
> - 基于 `esbuild` 的脚手架
> - 新年小玩具，真正像写 vue 3.0 一样写小程序

## 特性
  - [x] vue 3 单文件组件语法
    - [x] script setup
    - [x] cssVars
    - [x] pug
    - [x] css preprocess
    - [x] postcss， 默认使用 `Taro` 的 `postcss-pxtransform`
  - [x] 基于 `esbuild` 的快速编译脚手架
    - [x] 编译时宏函数，友好的类型提示和代码补全
